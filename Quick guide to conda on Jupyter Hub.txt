What to do:
--Open a terminal

Enable conda:
--Type to command below to enable conda and environments (will take a while the first time)
$source /cvmfs/ligo-containers.opensciencegrid.org/lscsoft/conda/latest/etc/profile.d/conda.sh

Add the previous comand to ".bash_profile":
--Type all the files/folders
$ls -a
--edit .bash_profile
$nano .bash_profile
--Paste "source /cvmfs/ligo-containers.opensciencegrid.org/lscsoft/conda/latest/etc/profile.d/conda.sh" (without quotation marks) to the end of the file
--Ctrl+X to exit
--Save changes (Y)

Create environment with all necessary libraries and activate it.
--Type command below to do create environment from the ylm file provided(might take a while the first time) (repleace "env1" with any name you feel like)
$conda env create --name env1 -f environmentIAA.yml
--Activate the environment you just created (again, replace env1 with whatever was the name of the environment you created)
--You may also want to add this to the .bash_profile file so you don't have to type this every time
$conda activate env1

Make sure Jupyter can now use the kernel of your environment.
--Run the code below to install ipykernel (needed)
$conda install -c anaconda ipykernel
--Run the code below to make your new environment kernel accessible through jupyter (again, replace env1 with whatever is the name for your environment).
$python -m ipykernel install --user --name env1 --display-name "Python3.7 (env1)"

Enabling buttons and controls on JupyterLab
--Run the following commands to install ipywidgets and nodejs
$conda install -c conda-forge jupyterlab
$conda install -c anaconda ipywidgets
$conda install -c conda-forge/label/gcc7 nodejs
